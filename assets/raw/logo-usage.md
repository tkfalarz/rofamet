# Logo Usage Notes — Rofamet

- Primary marks: `logo-gate-horizontal.svg` (horizontal lockup) and `logo-gate-stacked.svg` (stacked lockup).
- Colors: use tokens from `COLOR_PALETTE_PLAN.md`:
  - `--color-primary`: #F4F6F8 (background)
  - `--color-secondary`: #4B5563
  - `--color-accent`: #FF6B35
  - `--color-text`: #0F1724
- Clear space: 0.5× cap height around combined lockup.
- Minimum sizes:
  - Horizontal lockup: 80px width.
  - Stacked/icon-only: 32px square for UI; favicon must be legible at 16px.
 - Square lockup: use `logo-gate-square.svg` for avatar/lockup needs where a square container is required; prefer 256–512px exported PNGs for social and app stores.
- File deliverables to produce after approval: SVG source, PNG exports at 512/256/128/64/32/16, PDF.
- Accessibility: verify contrast for any color pair used; prefer `--color-on-accent` on `--color-accent` for reversed variants.
